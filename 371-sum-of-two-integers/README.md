# 两整数之和

## 描述

**不使用** 运算符 `+` 和 `-` ​​​​​​​，计算两整数 ​​​​​​​`a` 、`b` ​​​​​​​之和。

**示例 1:**

    
    
    **输入:** a = 1, b = 2
    **输出:** 3
    

**示例 2:**

    
    
    **输入:** a = -2, b = 3
    **输出:** 1



# Sum of Two Integers

## Description



Calculate the sum of two integers _a_ and _b_ , but you are **not allowed** to use the operator `+` and `-`.

**Example 1:**

    
    
    **Input:** a = 1, b = 2
    **Output:** 3
    

**Example 2:**

    
    
    **Input:** a = -2, b = 3
    **Output:** 1
