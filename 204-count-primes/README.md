# 计数质数

## 描述

统计所有小于非负整数  _n  _的质数的数量。

**示例:**

    
    
    **输入:** 10
    **输出:** 4
    **解释:** 小于 10 的质数一共有 4 个, 它们是 2, 3, 5, 7 。
    



# Count Primes

## Description



Count the number of prime numbers less than a non-negative number, **_n_**.

**Example:**

    
    
    **Input:** 10
    **Output:** 4
    **Explanation:** There are 4 prime numbers less than 10, they are 2, 3, 5, 7.
