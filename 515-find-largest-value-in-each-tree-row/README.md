# 在每个树行中找最大值

## 描述

您需要在二叉树的每一行中找到最大的值。

**示例：**

    
    
    **输入:** 
    
              1
             / \
            3   2
           / \   \  
          5   3   9 
    
    **输出:** [1, 3, 9]
