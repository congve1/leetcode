# 二叉树的中序遍历

## 描述

给定一个二叉树，返回它的 _中序  _遍历。

**示例:**

    
    
    **输入:** [1,null,2,3]
       1
        \
         2
        /
       3
    
    **输出:** [1,3,2]

**进阶:**  递归算法很简单，你可以通过迭代算法完成吗？
