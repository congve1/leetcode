# 有序数组的平方

## 描述

给定一个按非递减顺序排序的整数数组 `A`，返回每个数字的平方组成的新数组，要求也按非递减顺序排序。



**示例 1：**

    
    
    **输入：** [-4,-1,0,3,10]
    **输出：** [0,1,9,16,100]
    

**示例 2：**

    
    
    **输入：** [-7,-3,2,3,11]
    **输出：** [4,9,9,49,121]
    



**提示：**

  1. `1 <= A.length <= 10000`
  2. `-10000 <= A[i] <= 10000`
  3. `A` 已按非递减顺序排序。



# Squares of a Sorted Array

## Description



Given an array of integers `A` sorted in non-decreasing order, return an array of the squares of each number, also in sorted non-decreasing order.



**Example 1:**

    
    
    **Input:** [-4,-1,0,3,10]
    **Output:** [0,1,9,16,100]
    

**Example 2:**

    
    
    **Input:** [-7,-3,2,3,11]
    **Output:** [4,9,9,49,121]
    



**Note:**

  1. `1 <= A.length <= 10000`
  2. `-10000 <= A[i] <= 10000`
  3. `A` is sorted in non-decreasing order.
