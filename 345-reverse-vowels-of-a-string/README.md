# 反转字符串中的元音字母

## 描述

编写一个函数，以字符串作为输入，反转该字符串中的元音字母。

**示例 1:**

    
    
    **输入:** "hello"
    **输出:** "holle"
    

**示例 2:**

    
    
    **输入:** "leetcode"
    **输出:** "leotcede"

**说明:**  
元音字母不包含字母"y"。



# Reverse Vowels of a String

## Description



Write a function that takes a string as input and reverse only the vowels of a string.

**Example 1:**

    
    
    **Input:** "hello"
    **Output:** "holle"
    

**Example 2:**

    
    
    **Input:** "leetcode"
    **Output:** "leotcede"

**Note:**  
The vowels does not include the letter "y".
