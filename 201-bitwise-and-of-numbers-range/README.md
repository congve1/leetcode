# 数字范围按位与

## 描述

给定范围 [m, n]，其中 0 <= m <= n <= 2147483647，返回此范围内所有数字的按位与（包含 m, n 两端点）。

**示例 1:  **

    
    
    **输入:** [5,7]
    **输出:** 4

**示例 2:**

    
    
    **输入:** [0,1]
    **输出:** 0



# Bitwise AND of Numbers Range

## Description



Given a range [m, n] where 0 <= m <= n <= 2147483647, return the bitwise AND of all numbers in this range, inclusive.

**Example 1:**

    
    
    **Input:** [5,7]
    **Output:** 4
    

**Example 2:**

    
    
    **Input:** [0,1]
    **Output:** 0
