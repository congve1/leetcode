# 组合

## 描述

给定两个整数 _n_ 和 _k_ ，返回 1 ... _n_ 中所有可能的 _k_ 个数的组合。

**示例:**

    
    
    **输入:**  n = 4, k = 2
    **输出:**
    [
      [2,4],
      [3,4],
      [2,3],
      [1,2],
      [1,3],
      [1,4],
    ]

## Description
