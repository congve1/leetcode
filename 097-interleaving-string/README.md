# 交错字符串

## 描述

给定三个字符串  _s1_ , _s2_ , _s3_ , 验证  _s3_  是否是由  _s1_  和  _s2_ 交错组成的。

**示例 1:**

    
    
    **输入:** s1 = "aabcc", s2 = "dbbca", _s3_ = "aadbbcbcac"
    **输出:** true
    

**示例  2:**

    
    
    **输入:** s1 = "aabcc", s2 = "dbbca", _s3_ = "aadbbbaccc"
    **输出:** false
